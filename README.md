# Assignment 2: 2D Platformer Development

This is a 2D platformer game created in Unity for the Game Development assignment.

## Gameplay

The player starts at the start area and must move through a complete platforming level. The player can move left and right, jump, and double jump. The player must collect all coins before reaching the goal area.

## Features

- 2D player movement
- Jumping and double jumping
- Complete playable platformer level
- Start area and goal area
- Collectible coin system
- Goal requires all coins before completion
- Enemy lose condition
- Falling death zone
- Win and lose gameplay loop
- UI showing collected coins and game messages
- Redesigned visual style with colors and decorative background objects

## Controls

- A / D: Move left and right
- Left Arrow / Right Arrow: Move left and right
- Space: Jump
- Press Space again in the air: Double jump

## Built With

- Unity 6
- C#
- Universal 2D / URP
Final working version.
To run the game, open Assets/_Game/Scenes/MainScene in Unity and press Play.
